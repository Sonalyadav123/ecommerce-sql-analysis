# Ecommerce SQL Analysis Project

## Project Overview
This project demonstrates end-to-end SQL analysis on an ecommerce dataset.  
It includes business problems from basic to advanced level, similar to real data analyst job scenarios.

The project covers:
- Data cleaning
- Sales analysis
- Customer behavior
- Inventory insights
- Shipping performance
- Payment tracking

---

## Dataset Structure

### Tables Used
1. Products
2. Categories
3. Sellers
4. Customers
5. Orders
6. Order Items
7. Shipping
8. Payments
9. Inventory

## Business Problems Covered

### Basic Level
- Total orders
- Total revenue
- Top selling products
- Customer count by state
- Category wise sales

### Intermediate Level
- Monthly sales trends
- Top customers by revenue
- Seller performance
- Average order value
- Repeat customers

### Advanced Level
- Customer lifetime value
- Cohort analysis
- Product profitability
- Shipping delay analysis
- Inventory risk detection

Total: **70+ real-world SQL business problems**
